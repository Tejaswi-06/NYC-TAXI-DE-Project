## 📊 Dataset

The dataset used in this project is the NYC Taxi Trip Data (2023 – Green Taxi).

- Source: https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page  
- Data Format: Parquet (Green Taxi trip records) 

Data is ingested dynamically using Azure Data Factory and further transformed and processed using Azure Databricks.
